***GNU (Nano's ANOther editor)*** – edytor tekstu oparty na bibliotece ncurses dla systemów Unix i uniksopodobnych. Jest to wolny klon edytora `pico`, części programu pocztowego `pine`. Nano udostępnia funkcjonalność i skróty klawiaturowe `pine`, lecz nie umożliwia ścisłej współpracy z nim.  
Później Nano wprowadził ulepszenia, których brakowało jego pierwowzorowi, m.in. kolorowanie tekstu, wyszukiwanie przy pomocy wyrażeń regularnych oraz obsługę wielu buforów naraz.

___

| Skrót       | Opis                                                                                |
| ----------- | ----------------------------------------------------------------------------------- |
| `CTRL + A`  | Przeskakuje na początek wiersza.                                                    |
| `CTRL + E`  | Przechodzi na koniec linii.                                                         |
| `CTRL + Y`  | Przewija stronę w dół.                                                              |
| `CTRL + V`  | Przewija stronę w górę.                                                             |
| `CTRL + G`  | Wyświetla okno pomocy ze wszystkimi poleceniami.                                    |
| `CTRL + O`  | Zapisuje plik. Nano poprosi o podanie lub potwierdzenie nazwy pliku.                |
| `CTRL + W`  | Wyszukuje frazę w tekście. `ALT + W` ponawia wyszukiwanie tej samej frazy.          |
| `CTRL + K`  | Wycina zaznaczoną linię do bufora (jak do schowka).                                 |
| `CTRL + U`  | Wkleja tekst z bufora w wybranym miejscu.                                           |
| `CTRL + J`  | Justuje (wyrównuje) bieżący akapit.                                                 |
| `CTRL + C`  | Pokazuje pozycję kursora (wiersz, kolumna, znak).                                   |
| `CTRL + R`  | Wstawia zawartość innego pliku w bieżącej pozycji.                                  |
| `CTRL + X`  | Zamyka edytor. Pyta o zapis, jeśli dokonano zmian.                                  |
| `CTRL + \\` | Zastępuje ciąg znaków lub wyrażenie regularne.                                      |
| `CTRL + T`  | Uruchamia sprawdzanie pisowni (jeśli dostępne).                                     |
| `CTRL + _`  | Przechodzi do wybranego wiersza i kolumny.                                          |
| `ALT + A`   | Rozpoczyna zaznaczanie tekstu. Można połączyć z `CTRL + K`, aby wyciąć zaznaczenie. |
